# **Dashboard - Uber**

## English:
This repository holds an Uber analytics dashboard made in Power BI as a personal project, which can be downloaded in .pbix format.
It also holds the .csv file which contains the data from around 150 thousand trips during 2024.
* **Data source:** https://www.kaggle.com/datasets/yashdevladdha/uber-ride-analytics-dashboard?resource=download&select=ncr_ride_bookings.csv

## Español:
Este repositorio contiene un tablero de datos analítico de Uber hecho en Power BI, creado como proyecto personal, que se puede descargar en formato .pbix.
También contiene el archivo .csv que contiene los datos de alrededor de 150 mil viajes de Uber a lo largo de 2024.
* **Fuente de los datos:** https://www.kaggle.com/datasets/yashdevladdha/uber-ride-analytics-dashboard?resource=download&select=ncr_ride_bookings.csv
